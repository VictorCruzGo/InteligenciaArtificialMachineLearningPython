Tus respuestas

1. Queremos usar Machine Learning para descubrir si un paciente es diabético o no. Disponemos de estos datos para 500 pacientes, almacenados en dos tablas: una de pruebas analíticas con una fila por cada test realizado y otra de diagnósticos, con una fila por cada paciente.

 

¿Se pueden usar estos datos para determinar si un nuevo paciente es diabético usando Machine Learning?

2. Respecto al formato de los datos de la pregunta anterior, ¿están en un formato que permita usarlos en Machine Learning?

3. ¿Qué transformaciones o procesos deberíamos utilizar para preparar dichos datos para el aprendizaje?

4. Los tipos de cada campo que usamos en Machine Learning determinan cómo va a tratar los datos el algoritmo para aprender, y por lo tanto es importante que sean correctos.¿Cuáles son los tipos de cada uno de los campos que contiene el fichero Diabetes_es_ext.csv?

Para responder, podéis usar el servicio de BigML de forma totalmente gratuita. Los pasos a realizar serán:

- Registrarse en BigML (https://bigml.com/accounts/register/) proporcionando vuestro correo electrónico.

- Activar vuestra cuenta desde el enlace que recibiréis por correo electrónico.

Esto os permitirá acceder al Dashboard, donde podréis subir el fichero usando el botón del menú que mostramos en la imagen y escribiendo s3://bigml-public/csv/Diabetes_es_ext.csv en la casilla de texto.

Para descargarlo a vuestro ordenador podéis usar esta ubicación alternativa: https://static.bigml.com/csv/Diabetes_es_ext.csv

5. Utilizando el objeto Source generado en la pregunta anterior, deberéis asegurar que vuestros datos se interpretan correctamente verificando que el locale utilizado (que determina el separador decimal en los números) es el inglés, dado que el fichero que usamos tenia los datos en este formato. Desplegad el panel de configuración de Sources, tal como veis en la imagen.

 

Si vuestra pantalla coincide con la imagen, podéis seguir sin más al siguiente paso volviendo a plegar el panel de configuración. Si no es así (dependerá del idioma usado en el navegador), deberéis cambiar el valor a lo que muestra la imagen y pulsar el botón de update que actualizará vuestro Source y reinterpretará vuestros datos numéricos usando el punto como separador decimal.

Finalmente, construid un Dataset utilizando la acción 1-click dataset para analizar la distribución de los datos (os recomendamos que no borréis este Dataset, ya que se usará como punto de partida en preguntas posteriores). ¿Cuáles de estas afirmaciones son ciertas?
